Условия участия в тестировании

Leventime предоставляет возможность использования дополнительного функционала Сервиса, позволяющего в автоматическом режиме загружать в Сервис на Сайте, обновлять и продвигать Контекстные объявления с помощью специального файла - фида.

Параметры Контекстных объявлений, включая ставку стоимости каждого из них, указанные при заполнении такого фида, выгружаются в Рекламную кампанию в Сервисе. Изменение таких параметров недоступно на Сайте и может быть осуществлено исключительно через изменение фида. Применение изменений в Контекстных объявлениях на Сайте, загруженных через фид, производится при очередном обновлении Рекламной кампании, которое осуществляется с периодичностью не чаще, чем 1 раз в 6 часов, если включена опция обновления. Пользователь может устанавливать ставки для каждого Контекстного объявления в фиде, либо применить рекомендованные ставки, либо установить ставку для каждого Контекстного объявления в Сервисе, в Личном кабинете на Сайте.

Датой начала оказания Услуг считается дата фактической активации Контекстного объявления. При заполнении всех необходимых параметров и успешной выгрузке Контекстных объявлений из фида в Сервис, все Контекстные объявления автоматически активируются.
